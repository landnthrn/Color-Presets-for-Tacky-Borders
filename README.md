# 135 Color-Presets-for-Tacky-Borders

Tacky-Borders let you customize window borders with color animations, gradients, one side color fades, and static colors.   
Works on Windows 10 and 11.

## [GET TACKY-BORDERS HERE](https://github.com/lukeyou05/tacky-borders)

### Color Presets include  
- Clockwise Spirals  
- Counter Clockwise Spirals  
- Gradient Colors  
- One Sided Color Fade *(Left & Right)*  
- Static Colors  

Comes with two speed variations for the spiral animations â€” Default & Slower

`Color Map.txt` ~ Displays all the hex codes used in the color variations  
`How to Use + Info` ~ Simple guide on how to use 

### NOTE:

I'm on Windows 10 and Tacky Borders works just fine on it, I fixed a couple bugs though, not sure if they were due to Windows 10 or not.  

- There was a border across each of my monitors, it was thinking my monitor screens were a window or something.  

- And being on a dual monitor setup whenever I fullscreened something on one window it would bleed over a little to other monitor.  
*(Probably shouldn't call this a bug cause it was just a matter of thickness)*  

Both are fixed for all the color presets.
